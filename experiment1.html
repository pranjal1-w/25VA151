HTML

HTML, or HyperText Markup Language, is the standard markup language used to create and structure content on the web. It defines the structure and meaning of web pages by using elements, commonly known as tags, to enclose and format text, images, links, and other media. These elements are delineated by tags written using angle brackets, such as for headings and for paragraphs. HTML is the foundational building block of the web, working alongside CSS for styling and JavaScript for interactivity.

TYPE OF HTML ELEMENTS

  1. normal elements
  2. raw text elements
  3. void elements

USES OF HTML

foundation use
foundational technology for creating and structuring content on the web, serving as the primary building block for websites and web applications.
structure uses
t defines the structure and meaning of web content, enabling browsers to display text, images, links, and other media
creating uses
HTML is used to create web pages and web documents, providing a standardized way to organize content through elements like headings, paragraphs, lists, and tables.

CSS

CSS stands for Cascading Style Sheets, a style sheet language used to describe the presentation and styling of a document written in a markup language such as HTML or XML. It enables the separation of content (structured with HTML) from its visual design, allowing control over layout, colors, fonts, spacing, and responsiveness across different devices. CSS works by applying rules to HTML elements through selectors, with styles cascading in a hierarchical order based on specificity and source order. This separation improves maintainability, enhances performance through reusable style files, and supports responsive design using tools like media queries and Flexbox.

TYPE OF CSS

  1. inline
  2. internal
  3. external

USES OF CSS

styling web pages
styling web pages by defining fonts, colors, layouts, and other visual elements, which enhances the visual appeal and readability of websites.
responsive design
CSS plays a crucial role in responsive design, allowing web pages to adapt to different screen sizes and devices through media queries, ensuring a consistent user experience across desktops, tablets, and mobile phones.

JAVA SCRIPT

JavaScript is a high-level, interpreted programming language primarily used to create dynamic and interactive content on web pages. It is a core technology of the web, alongside HTML and CSS, and is used to manipulate webpage content through the Document Object Model (DOM), handle user interactions, and enable real-time updates without requiring full page reloads. Originally developed by Brendan Eich at Netscape in 1995 as LiveScript, it was renamed JavaScript to leverage the popularity of Java at the time. JavaScript is multi-paradigm, supporting event-driven, functional, and object-oriented programming styles, and is now used not only in web browsers but also on servers via runtimes like Node.js, Deno, and Bun.
